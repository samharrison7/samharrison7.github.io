procedure, private :: calculateDepth => calculateDepth1

  • private pure function calculateDepth1(me, W, S, Q) result(r)

    Calculate water depth from Manning's roughness coefficient, using Newton's method: $$ D_i = D_{i-1} - \frac{f(D_{i-1})}{f'(D_{i-1})} $$ where $$ f(D) = WD \left( \frac{WD}{W+2D} \right)^{2/3} \frac{\sqrt{S}}{n} - Q = 0 $$ and $$ f'(D) = \frac{\sqrt{S}}{n} \frac{(DW)^{5/3}(6D + 5W)}{3D(2D + W)^{5/3}} $$
